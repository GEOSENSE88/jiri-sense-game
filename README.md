# 🗺️ 지리SENSE GAME (v5)
**제작: 산남고 이성훈**

수능 한국지리를 준비하는 학생을 위한 **시·군 단위 백지도 경쟁 퀴즈 웹 게임**입니다.
**2027 수능특강 + 수능완성** 한국지리와 **최근 5개년(2021.03~2026.06) 고3 기출 31회분 출제 빈도 분석**을 바탕으로 제작했습니다.

## 🔥 기출 빈도 연동 (v5)
- 5개년 고3 학평·모평·수능 31회분 PDF를 전수 분석해 **지역별 언급 빈도**를 산출 (`_extract\freq_analysis.py`)
  - 동음이의어 보정: '상주인구'≠상주시, '영동 지방'≠영동군, 바다 동해·남해 제외 등
- **빈출 지역일수록 더 자주 출제** — 위치 사냥·지역 판독·마스코트 모드에 가중 무작위 출제 적용
- 홈 화면에 **빈출 지역 TOP 12** 랭킹 표시 (울산 42회, 제주 38회, 태백 18회, 울릉 16회…)
- 정답 해설과 탐색 모드에 **기출 N회 언급 + 출제 경향 노트(31개 빈출 지역) + 📷 네이버 이미지 자료 검색 버튼** 표시
- 탐색 모드에서 빈출 지역은 점이 더 크게 표시됨

## 앱 아이콘·설치
`icon-512/192.png`, `apple-touch-icon.png`, `favicon.ico`, `manifest.json` 포함 —
모바일 브라우저에서 **"홈 화면에 추가"** 하면 앱처럼 설치되어 아이콘과 함께 실행됩니다.

## 실행 방법
- **PC**: `index.html` 더블클릭 (설치·인터넷 불필요)
- **모바일(권장 배포)**: 폴더를 [Netlify Drop](https://app.netlify.com/drop)이나 GitHub Pages에 올리면
  주소 하나로 학급 전체가 스마트폰에서 접속할 수 있습니다.
  세로 화면에 맞춘 반응형 UI + 터치(탭·핀치 줌·드래그 팬)를 지원합니다.
> 점수·숙련도·랭킹은 각 기기 브라우저(localStorage)에 저장됩니다.

### 수능완성 반영 내용 (v3)
독도(위치·구성·형성 시기), 국토 정중앙(양구), 높새바람, 철원 오대쌀·강릉 초당 순두부·인제 황태,
삼척 환선굴, 수도권 특례시(수원·고양·용인·화성)와 인구 100만 도시, 수도권 제조업 출하액(화성 1위),
주간 인구 지수, 전남도청(무안 남악), 영암·해남 관광 레저형 기업도시, 한라산 화산체 구조, 고팡 등.
마스코트는 교재 수록(한우리·머돌이와 머순이·빛돌이·배돌이·무안 양파) + 지자체 공식 홈페이지 검증분만 수록.

## 경쟁·학습 요소
- **점수**: 기본 점수 + 남은 시간 보너스 + 콤보 보너스 (콤보당 +10)
- **계급**: 누적 XP에 따라 "지리 새내기 → … → 한국지리 그랜드마스터" 9단계 승급
- **지역별 숙련도**: 권역별 정답률을 추적해 취약 지역을 한눈에 확인
- **명예의 전당**: 모드별 최고 점수 랭킹 (TOP 10 저장)
- **출제 범위 필터**: 홈 화면에서 권역을 선택하면 해당 권역만 집중 출제

## 파일 구성
- `index.html` / `style.css` — 화면 (모바일: 문제→지도→보기 세로 배치)
- `game.js` — 게임 엔진 (터치 제스처·줌, 기후 그래프 SVG 렌더러 포함)
- `questions.js` — 문제 은행 (MCQ 152 / OX 78 / 시·도 30 / 마스코트 15) ← **문항 추가는 이 파일에**
- `map-data.js` — 시·군 161개 + 시·도 외곽선 SVG, 지점 103개(소속 시·군 매핑 포함)
- `stats-data.js` — 기후 관측소 51곳 월별 기온·강수(평년값) + 시도별 통계청 자료(순위 포함)
- `_test_data.js`, `_test_runtime.js`, `_test_climate.js` — 개발용 자동 테스트 (`node _test_data.js`)

## 문항 추가 방법
`questions.js` 의 배열에 같은 형식으로 객체를 추가하면 즉시 반영됩니다.
```js
// 4지선다
{region:'호남', q:'질문?', choices:['①','②','③','④'], answer:0, exp:'해설'},
// OX
{region:'제주', q:'명제.', answer:true, exp:'해설'},
```
위치 지점 추가는 `map-data.js`의 `LOCATIONS` 에 `{name, x, y, region, fact}` 형식으로 추가하세요.
